# Rumor Detection and Analysis on Tweets
1) Wrote Python scripts to scrape tweets using tweepy API. 
2) Developed a pipeline to conduct data preprocessing, feature engineering and rumor classification on tweets, using  BERT model from PyTorch and ensemble classifier XGBoost. Ran 35k+ raw data with 60% labelled via GPU. Achieved 90% F1 score in the test dataset and ranked top 25% of the Kaggle competition.
3) Conducted rumor detection on 10k COVID-related tweets. Analysed topic trends, hashtags, sentiment and user traits in Python.
